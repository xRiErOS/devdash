# BreadcrumbItem

`status:stable` · Molecule · Cluster `04.20 Navigation`

## Zweck

Ein Segment eines Breadcrumb-Pfads: optionales führendes Icon + Label als Link
(`href`) bzw. als aktuelles, nicht-verlinktes Segment (`current`), gefolgt von
einem Separator-Icon (Chevron) — außer beim letzten Segment (`isLast`).
Komponiert das `Link`-Atom; der Separator ist ein dekoratives Icon.

Dumb-Molecule (CONV-molecule-boundary): präsentational, kein Routing-State. Der
konsumierende Organismus (Breadcrumb) reiht mehrere BreadcrumbItem und setzt
`current`/`isLast` am letzten.

## Wann verwenden

- **Ja:** Hierarchie-Pfad (Home › Sprints › DD-1) im Seitenkopf.
- **Nein:** flache Tab-Navigation → `TabButton`. Sidebar-Eintrag → `NavItem`.

## Zustände

Verlinktes Segment (gedämpfter Link mit Separator) vs. aktuelles Segment
(`current` → Klartext, `aria-current="page"`, kein Link). `isLast` unterdrückt
den Separator.

<Canvas of={BreadcrumbItemStories.Variant_States} />

## Barrierefreiheit

- Verlinkte Segmente sind native `<a href>` (Tastatur/Fokus gratis); das aktuelle
  Segment ist Klartext mit `aria-current="page"` (kein toter Link).
- Der Separator-Chevron ist `aria-hidden` (rein dekorativ).
- Der konsumierende Organismus klammert die Items in `<nav aria-label>`.
