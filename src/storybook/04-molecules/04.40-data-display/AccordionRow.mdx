# AccordionRow

`status:stable` · Molecule · Cluster `04.40 Data Display`

## Zweck

Eine Disclosure-Zeile: Trigger-Button (optionales Lead-Icon + Titel + Chevron)
und ein Panel-Slot, der nur bei `open` rendert. Der Baustein für aufklappbare
Abschnitte (komponiert von OR-22, Children-Accordion).

Dumb-Molecule (CONV-molecule-boundary): controlled — `open` ist Prop, `onToggle`
der Callback. Den State besitzt der Consumer/Organismus (OR-22); so lassen sich
mehrere Rows koordinieren (Single-Open-Accordion).

## Wann verwenden

- **Ja:** aufklappbarer Abschnitt mit Trigger-Zeile (FAQ, Detail-Gruppen, DoD).
- **Nein:** Tab-Wechsel → `TabButton`. Modal-Overlay → `ModalShell`. Reine
  Label/Wert-Zeile → `MetaRow`.

## Zustände

`open` schaltet das Panel frei und dreht den Chevron (180°). `disabled` sperrt
den Trigger. Ohne `open` ist das Panel nicht im DOM.

<Canvas of={AccordionRowStories.Variant_States} />

## Barrierefreiheit

- Nativer `<button>` mit `aria-expanded`; `aria-controls`/`id` koppeln Trigger
  und Panel (`role="region"`).
- Der Chevron ist `aria-hidden` (Dekoration) — die Bedeutung trägt der Titel.
- `disabled` setzt das native disabled-Attribut (Fokus + Klick blockiert).
