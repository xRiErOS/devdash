# Button

`status:stable` · Atom · Cluster `03.10 Inputs`

## Zweck

Token-saubere Action-Primitive. Trägt jede klickbare Aktion mit semantischer
Akzentrolle (DD-47). Props-driven, kein Store — komponierbar in Molecules/Organisms.

## Wann verwenden

- **Ja:** ausgelöste Aktion (Speichern, Anlegen, Abbrechen, Löschen).
- **Nein:** Navigation → `Link`. Reines Icon ohne Label → `IconButton`.
  Mehrfachauswahl/Toggle → `TabButton` / `Checkbox`.

## Barrierefreiheit

- Natives `<button type>` — Fokus + Enter/Space gratis.
- `loading` setzt `aria-busy` und blockt Doppel-Klick (disabled).
- Icons sind `aria-hidden`; das Label trägt die Bedeutung. Icon-only → `IconButton`
  mit `aria-label`.
- Kontrast: Akzent-Tokens sind WCAG-AA gegen `--on-accent` geprüft.
