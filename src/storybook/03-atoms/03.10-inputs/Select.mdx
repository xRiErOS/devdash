# Select

`status:stable` · Atom · Cluster `03.10 Inputs`

## Zweck

Token-saubere Dropdown-Auswahl — drittes Inputs-Atom der Trias Text (`Input`) /
Boolean (`Checkbox`) / Dropdown (`Select`). Gestyltes Native-`<select>`
(appearance-none + Custom-Chevron), props-driven, kein Store.

## Wann verwenden

- **Ja:** Auswahl genau eines Werts aus einer kurzen, festen Liste (Priorität,
  Typ, Status-Filter).
- **Nein:** Boolean → `Checkbox`. Freitext → `Input`. Mehrfachauswahl oder
  lange/suchbare Liste → eigenes Combobox-Molecule (nicht dieses Atom).

## Barrierefreiheit

- Natives `<select>` — Tastatur-Navigation + mobiler Native-Picker gratis.
- `invalid` setzt `aria-invalid=true`; Fehlertext gehört ans umgebende Feld-Molecule.
- Chevron + leadingIcon sind dekorativ (`pointer-events-none`); die Optionen
  tragen die Bedeutung.
- Kontrast: `--surface0`/`--text` WCAG-AA, Danger-Outline aus `--accent-danger`.
