# Switch

`status:stable` · Atom · Cluster `03.10 Inputs`

## Zweck

Boolean-Toggle für einen sofort wirksamen An/Aus-Zustand — z. B. ein Feature
ein-/ausschalten. Token-saubere Track-/Thumb-Primitive auf einem nativen
`button`-Element.

## Wann verwenden

- **Ja:** binäre Einstellung, die sofort greift (kein Speichern-Klick nötig).
- **Nein:** Auswahl, die erst mit einem Absende-Schritt wirksam wird → `Checkbox`.
  Mehr als zwei sich ausschließende Optionen → `TabButton` / `Select`.

Der Unterschied zur `Checkbox`: ein `Switch` schaltet unmittelbar, eine
`Checkbox` markiert eine Auswahl, die typischerweise erst beim Absenden zählt.

## Zustände

`checked` (Track = `--accent-primary`, Thumb rechts), `disabled` (`opacity` +
`cursor-not-allowed`), `size` (`sm` / `md`). Das optionale `label` steht rechts.

## Barrierefreiheit

- Natives `button` mit `role="switch"` — Fokus und `Space` / `Enter` gratis.
- `aria-checked` spiegelt den `checked`-Wert (`true` / `false`).
- `disabled` setzt das native Attribut — der Toggle ist nicht fokussierbar/klickbar.
- Der Thumb ist `aria-hidden`; die Bedeutung trägt der `role="switch"`-Zustand
  und das optionale `label`.
