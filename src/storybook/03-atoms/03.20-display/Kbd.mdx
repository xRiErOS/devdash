# Kbd

`status:stable` · Atom · Cluster `03.20 Display`

## Zweck

Tastatur-Shortcut-Anzeige (keyboard-first) auf nativem `<kbd>`. Mono, getönte
Fläche (`--surface1`), feine Border (`--surface2`), gerundet, klein. Props-driven,
token-clean.

## Wann verwenden

- **Ja:** Anzeige einer Taste oder eines Tastaturkürzels (`S`, `Esc`, `⌘`) in
  Tooltips, Menüs, Hilfe-Hints — passend zum keyboard-first-Charakter des Tools.
- **Nein:** klickbarer Button → `Button`/`IconButton`. Status-/Kategorie-Chip →
  `Badge`/`Pill`.

## Zustände

`size` `sm`/`md`. Für eine Tastenkombination (z.B. `⌘`+`S`) mehrere `<Kbd>`
nebeneinander setzen — der Trenner (`+`) liegt beim Aufrufer.

<Canvas of={KbdStories.Sizes} />

## Barrierefreiheit

- Natives `<kbd>` trägt die semantische Bedeutung „Tastatureingabe" — keine
  ARIA-Rolle nötig.
- Kontrast: `--subtext0`-Text gegen `--surface1`-Fläche; Border `--surface2`
  grenzt visuell ab.
- Bei Bedarf `aria-label` über `...rest` setzbar (z.B. `aria-label="Speichern"`).
