# Pill

`status:stable` · Atom · Cluster `03.20 Display`

## Zweck

Kompakter Badge für Status, Tags, Zähler, Kategorien. Token-sauber, props-driven.

## Wann verwenden

- **Ja:** kurzer Label-/Status-Marker (Issue-Status, Tag, Rollen-Chip).
- **Nein:** klickbare Aktion → `Button`. Icon-Marker → `Icon`. Längerer Fließtext.
