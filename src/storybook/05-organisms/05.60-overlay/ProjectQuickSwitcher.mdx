# ProjectQuickSwitcher

Projekt-Picker mit Fuzzy-Search (Name/Slug/Prefix) und Keyboard-Navigation
(Arrow/Enter/Esc) über eine gefilterte Projekt-Liste plus zwei gepinnte Aktionen
(Projekt-Übersicht, Neues Projekt anlegen). Komponiert die Atoms Input und PopoverPanel.

Präsentational — kein fetch/Store/Routing. Die Mutation macht der Consumer über die
gehobenen Callbacks `onSelect`, `onOpenOverview`, `onCreateNew`, `onClose`.
