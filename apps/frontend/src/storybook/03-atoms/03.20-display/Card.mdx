# Card

`status:stable` · Atom · Cluster `03.20 Display`

## Zweck

Flächen-/Panel-Primitive mit Token-Ebenen-Hierarchie (`tone`). Container für
geschachtelte Inhalte (Pattern P08), props-driven.

## Wann verwenden

- **Ja:** abgegrenzte Inhaltsfläche (Listeneintrag, Panel, Tile).
- **Nein:** Seiten-Layout/Grid → Layout-Tier. Interaktive Karte mit Aktionen →
  Card + komponierte Atome im Molecule.

## Zustände

Tone (`crust`/`mantle`/`base`/`surface0` — Flächen-Tiefe), `bordered`, Padding
(`none`/`sm`/`md`, liest die responsiven `--space-*`-Tokens).
