# DependencyWidget

`status:stable` · Organism · Cluster `05.30 Widgets` · OR-19 · GF-254

## Zweck

Zeigt die Abhängigkeiten einer Entität in zwei Sektionen: **Vorgänger** (blockers,
„hängt ab von") und **Nachfolger** (blocked_by, „blockiert"). Jedes Item ist klickbar
(`onNavigate` → Dependency-Graph / Detail) und optional entfernbar (`onRemove`, unlink).

## Ebene-6-Capability-Gegencheck (A4)

OR-19 = 6 Caps: `dependency-list`, `dependency-graph`, `milestone-read`,
`milestone-dep-add`, `milestone-dep-list`, `milestone-dep-remove` (`_ebene6.md`).
Anzeige + Navigation hier; das Erfassen neuer Kanten ist `DependencyForm` (OR-20, GF-255).

## Wann verwenden

- **Ja:** Abhängigkeits-Region einer Detail-Ansicht (Issue oder Milestone).
- **Nein:** neue Abhängigkeit erfassen → `DependencyForm`. Voller Graph → Graph-View.

Nur Vorgänger · beide Sektionen · removable (mit Unlink-Trigger) · leer (Hinweis).
