# Grid

`status:stable` · Layout-Primitiv · `02.10 Primitives`

## Zweck

Responsives Auto-Fit-Raster. Die Spaltenzahl ergibt sich aus der minimalen
Spaltenbreite (`minmax(min, 1fr)` plus `auto-fit`) — kein Breakpoint-Gefummel,
keine fixen Spaltenzahlen.

## Wann verwenden

- **Ja:** Karten-/Kachel-Raster, das je Platz mehr oder weniger Spalten zeigt.
- **Nein:** asymmetrisches Inhalt-plus-Seitenspalte → `Sidebar`. Einzelne Reihe → `Cluster`.

## Props

`min` (minimale Spaltenbreite, rem — kein px, D10-C), `gap` (none/xs/sm/md/lg → `--space-*`).

## Reflow-Vertrag (D10-B/C)

Grid reflowt INTRINSISCH: Spalten werden hinzugefügt/entfernt, sobald je `min`-Breite
eine weitere Spalte passt — gesteuert vom Container, nicht vom Viewport. Bei
Unterschreiten von `min` fällt das Raster auf eine Spalte (Stack-artig). Breiten sind
fluid (`1fr`/`minmax`), keine fixen px-Spaltenbreiten.
