# MetaBlock

`status:stable` · Molecule · Cluster `04.40 Data Display`

## Zweck

Label/Value-Slot-Paar mit Top-Border-Trenner (Label über Wert) — für gestapelte
Meta-Listen in Detail-Seitenleisten. Der Value-Slot nimmt jeden ReactNode
(Klartext, `EntityPill`, `CopyButton`).

Dumb-Molecule (CONV-molecule-boundary): reine Display-Props, kein State.
Komponente bleibt geteilt in `atoms/`, Island-Story neu (D06).

## Wann verwenden

- **Ja:** gestapelte Label/Wert-Blöcke mit Trenner (Sidebar-Meta-Listen).
- **Nein:** kompakte 2-Spalten-Zeile bzw. Grid → `MetaRow`. Card-Shell →
  `MetaCard`. Chip-Grid → `MetaFieldGrid`.

## Zustände
