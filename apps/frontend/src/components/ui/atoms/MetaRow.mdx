# MetaRow

`status:stable` · Molecule · Cluster `04.40 Data Display`

## Zweck

Zweispaltige Label+Value-Zeile (Label 80px, Value 1fr) mit optionalem Color-Dot.
Der kompakte Baustein für Stammdaten-Übersichten — 2×n-fähig im Grid. Der
Value-Slot nimmt jeden ReactNode (Klartext, `EntityPill`, `CopyButton`).

Dumb-Molecule (CONV-molecule-boundary): reine Display-Props, kein State. Die
Komponente bleibt geteilt in `atoms/` (Consumer `MetaCard`, `MetaFieldGrid`),
bis sie selbst migriert ist (D06).

## Wann verwenden

- **Ja:** kompakte Label/Wert-Paare in Detail-Übersichten, Sidebars, Grids.
- **Nein:** gestapelter Block mit Trenner → `MetaBlock`. Fertige Karte mit
  Card-Shell → `MetaCard`. Chip-Grid mit Copy-for-AI → `MetaFieldGrid`.

## Zustände

Der `dot`-Schalter blendet einen Color-Dot vor dem Value ein (Farbe via
`--dot-color`, token-clean per className). Der Value-Slot ist beliebig
komponierbar.

## Barrierefreiheit

- Der Color-Dot ist rein dekorativ (kein Text) — die Bedeutung trägt stets der
  Value-Text; Farbe ist nie das alleinige Signal.
- Komponierte Slots (EntityPill, CopyButton) bringen ihre eigene a11y mit
  (aria-label am CopyButton, title an der EntityPill).
