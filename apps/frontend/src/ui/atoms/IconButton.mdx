# IconButton

`status:open` · Atom · Cluster `02 ATOMS/IconButton`

## Kurzbeschreibung

Icon-only-Aktion (Tool/Toggle) — Edit, Close, Filter, Expand. Das meistgenutzte
Atom der Shell.

## Zweck

Rendert ein echtes `<button>` mit einem Registry-Icon (inherit-Farbe). `on`
markiert den aktiven Zustand (Akzentfläche). Zwei Größen (`md` 30px, `sm` 20px).
Da Icon-only bedeutungstragend ist, ist `label` Pflicht (aria-label + title).

## Wann verwenden

- **Ja:** Werkzeug-Buttons ohne Text (Toolbar, Widget-Header, Panel-Toggle).
- **Nein:** Aktion mit Text → `Button`. Filter-Pill → `Chip`.

## Barrierefreiheit

### ARIA
`<button>` mit `aria-pressed={on}`; `label` → `aria-label` (Icon ist sonst
`aria-hidden`).

### Keyboard
Enter/Space lösen aus.
