# Textarea

`status:open` · Atom · Cluster `02 ATOMS/Textarea`

## Kurzbeschreibung

Mehrzeiliges Text-Eingabe-Atom — das vertikale Pendant zu `Input`.

## Zweck

Token-saubere, props-driven Eingabe (DD-56 Harvest). Vertikaler Resize-Handle
(`resize-y`), `text-base` (16px) gegen iOS-Zoom auf Focus. Kein Store/Fetch.

## Wann verwenden

- **Ja:** mehrzeilige Texte (Beschreibung, Notizen, Refinement).
- **Nein:** einzeilig → `Input`. Suchfeld mit Icon → `SearchField` (Molecule).
