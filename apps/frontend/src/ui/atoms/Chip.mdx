# Chip

`status:open` · Atom · Cluster `02 ATOMS/Chip`

## Kurzbeschreibung

Kleines, klickbares Filter-/Auswahl-Pill (z.B. Prioritäten P1–P4 im Filter-Menü).

## Zweck

Atom mit echtem `<button>` (kein rohes Element im Consumer). Props-driven;
`active` markiert die Auswahl (`aria-pressed`).

## Wann verwenden

- **Ja:** Mehrfach-/Einfach-Filter, Tag-Auswahl in einem Menü.
- **Nein:** Entitäts-ID → `EntityId`. Status → `StatusDot`. Aktion → `Button`.

## Barrierefreiheit

### ARIA

Fokussierbar; Enter/Space lösen aus (native Button-Semantik).
