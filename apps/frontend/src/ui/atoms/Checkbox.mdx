# Checkbox

`status:open` · Atom · Cluster `02 ATOMS/Checkbox`

## Kurzbeschreibung

Abhakbares Kontrollkästchen mit optionalem Label — z.B. für Akzeptanzkriterien.

## Zweck

Echtes `<button role="checkbox">`, props-driven. Der Haken im abgehakten Zustand
ist das Registry-Icon `check` (inherit-Farbe auf grüner Box) — kein CSS-Pseudo-
Element, damit der Stil utility-only bleibt.

## Wann verwenden

- **Ja:** Einzel-Toggle in Listen (Checklist, Akzeptanzkriterien).
- **Nein:** mehrere exklusive Optionen → `SegmentedControl`. Filter → `Chip`.
