# Kbd

`status:open` · Atom · Cluster `02 ATOMS/Kbd`

## Kurzbeschreibung

Visuelle Tastatur-Taste für Shortcut-Anzeigen (`⌘K`, `esc`, `⇧⌘P`).

## Zweck

Reines Display-Atom, props-driven. Ein Glyph je Instanz — Kombinationen entstehen
durch Aneinanderreihen mehrerer `Kbd` (siehe Story `Shortcuts`).

## Wann verwenden

- **Ja:** Shortcut-Hinweis in Command-Bar, Command-Palette, Menü-Zeile.
- **Nein:** klickbarer Filter/Tag → `Chip`. Entitäts-ID → `EntityId`.
