# Input

`status:open` · Atom · Cluster `02 ATOMS/Input`

## Kurzbeschreibung

Einzeiliges Text-Eingabe-Atom — das Pendant zu `Textarea`.

## Zweck

Token-saubere Form-Eingabe, props-driven. `text-base` (16px) verhindert iOS-Zoom
beim Fokus. Such-/Filter-Felder mit führendem Icon sind eine eigene Molecule
(`SearchField`), nicht dieses Atom.

## Wann verwenden

- **Ja:** einzeiliges Textfeld in Formularen (Titel, Wert).
- **Nein:** mehrzeilig → `Textarea`. Suchfeld mit Icon → `SearchField` (Molecule).
