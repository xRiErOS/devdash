# CONTEXT — cli-go / TUI (DD2)

TUI-Welt: das interaktive Cockpit (Bubble Tea). Datei-/Layout-Regeln → `apps/cli-go/CLAUDE.md`. Übergreifende Begriffe → Wurzel-`CONTEXT.md`.

- **Command View / Command Center** — die projektweite Issue-Sicht, das zentrale Kommando-Panel der TUI.
- **Browse-Tree** — die Primat-Sicht als Baum: Meilenstein → Sprint → Issue.
- **Detail-Accordion** — die editierbare Detail-Fläche mit fokussierbaren, aufklappbaren Sektionen (Meilenstein/Sprint/Issue teilen sich die Maschine).
- **Review-Cockpit** — die Sicht auf einen einzelnen Review-Sprint, in der der PO die Ergebnisse abnimmt.
- **Lobby** — der Einstieg mit der Projektauswahl (Home).
- **One-Shot vs. Cockpit** — Dual-Layer: scriptbare Einzelbefehle gegenüber der interaktiven TUI.
- **Palette** — das schwebende Kommando-/Aktions-Menü.
